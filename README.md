TypeScript curry example
===========================

Quick Start:
------------

```bash
# download it
$ git clone git@github.com:hrajchert/ts-curry-example.git
# install deps
$ cd ts-curry-example
$ npm install
# run it or test it
$ npm run start
$ npm t
```

Created with the [typescript-node-starter](https://github.com/acamica/typescript-node-starter)
